A 3-Minute Interactive Warm-Up Before You Present
StageReady is a production-level iOS app built with SwiftUI (iOS 17+) that helps students reset confidence before presentations, interviews, or viva sessions — in under 3 minutes.
Fully offline. No backend. Lightweight (<25MB). Smooth, modern, animation-driven experience.
✨ Features
🎙 Match the Wave
Live voice control challenge using AVAudioRecorder with animated wave metering and stability score visualization.
⏱ Clarity Speed
Tongue-twister timing challenge with intelligent pacing feedback and animated circular timer.
👁 Eye Focus
Camera-free concentration drill with tap accuracy scoring and animated focus arc.
🧠 Memory Spark
Short-term recall challenge with animated distraction and word-level error highlighting.
🎨 Design & UX
SwiftUI-only implementation
Clean MVVM architecture
Custom animated background
3-minute animated progress ring
Swift Charts integration
Smooth micro-interactions & hero transitions
Gradient glow buttons & rounded cards
Haptic feedback
Stunning Light & Dark mode (manual toggle with @AppStorage)
Minimal. Calming. Energetic. International-level polish.

Built to help students step on stage with clarity, control, and confidence — in just three minutes.
